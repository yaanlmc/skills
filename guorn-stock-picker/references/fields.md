### 筛选条件

格式：指标 + 运算符 + 范围 + 值

示例：
- 预期连续两年亏损 等于 0
- 收盘价 排名%最小 全部 35%
- 总市值 排名%最小 全部 29%

### 排名条件

格式：指标 + 从X到Y + 范围 + 值

示例：
- 收盘价 从小到大 一级行业内 30
- 市盈率(PE) 从小到大 一级行业内 30

## 关键 JavaScript 选择器

```javascript
// 查找登录按钮
document.querySelectorAll('a').forEach(el => {
    if (el.innerText.trim() === '登录') el.click();
});

// 查找手机号输入框
document.querySelectorAll('input').forEach(inp => {
    if (inp.placeholder?.includes('手机') && inp.offsetParent !== null) {
        inp.focus();
        inp.click();
    }
});

// 查找密码输入框
document.querySelectorAll('input[type="password"]').forEach(inp => {
    if (inp.offsetParent !== null) {
        inp.focus();
        inp.click();
    }
});

// 滚动到底部
window.scrollTo(0, document.body.scrollHeight);

// 滚动到顶部
window.scrollTo(0, 0);
```
