# 页面字段映射参考

## 登录页面

| 字段 | 选择器/定位方式 | 说明 |
|------|----------------|------|
| 登录按钮 | `innerText === '登录'` | 顶部导航栏 |
| 手机号输入框 | `input[placeholder*="手机"]` | 需要键盘模拟输入 |
| 密码输入框 | `input[type="password"]` | 需要键盘模拟输入 |
| 提交按钮 | `button` containing `登录` | 表单提交 |

## 账户总览页面

URL: `https://ycgr.fcsc.com/trader/home?id=<账户ID>&page=holdings`

| 字段 | 提取方式 | 示例值 |
|------|---------|--------|
| 当前净值 | 正则: `当前净值[\s\S]*?(\d+\.\d+)` | 1.5816 |
| 净值涨跌 | 正则: `\(-?[\d.]+%\)` | (-0.59%) |
| 总资产 | 正则: `总资产[\s\S]*?([\d,]+\.\d{2})` | 375,591.54 |
| 持仓市值 | 正则: `持仓市值[\s\S]*?([\d,]+\.\d{2})` | 268,098.00 |
| 可用现金 | 正则: `可用现金[\s\S]*?([\d,]+\.\d{2})` | 107,493.54 |

### 策略表格

## 策略详情页面

URL: `https://ycgr.fcsc.com/stock/strategy?sid=<策略ID>`

### 策略概览

| 字段 | 正则匹配 |
|------|---------|
| 策略名 | 页面顶部标题 |
| 创建日期 | `创建日期[:\s]*(\d{4}/\d{2}/\d{2})` |
| 下个调仓 | `下个调仓[:\s]*(\d{4}/\d{2}/\d{2})` |
| 调仓时点 | `调仓时点[:\s]*(\d+:\d+)` |
| 持仓股票数 | `持仓股票[:\s]*(\d+)只` |

### 收益统计

| 字段 | 正则匹配 |
|------|---------|
| 年化收益 | `年化收益[^\d]*([\d.]+)%` |
| 夏普比率 | `夏普比率[^\d]*([\d.]+)` |
| 最大回撤 | `最大回撤[^\d]*([\d.]+)%` |
| Alpha | `Alpha[^\d]*([\d.]+)%` |

### 持仓明细

表格结构：
- 序号
- 股票名称（含代码）
- 行业分类
- 买入日期
- 买入价格
- 最近收盘价
- 当前仓位
- 累计涨幅

### 调仓指令

| 字段 | 说明 |

| 参考价 | 调仓参考价格 |

## 常见问题

### React 表单输入问题

一创果仁网使用 React 框架，直接设置 `input.value` 不会触发 onChange 事件。

解决方案：使用 CDP `Input.dispatchKeyEvent` 模拟键盘输入。

```javascript
// 错误方式
input.value = "18080155660";

// 正确方式（通过 CDP）
for (char in text) {
    Input.dispatchKeyEvent({ type: "keyDown", text: char });
    Input.dispatchKeyEvent({ type: "keyUp", text: char });
}
```

### 策略 ID 格式

策略 ID 格式：`105503.R.xxxxxxxxxxxxxxx`

示例：`sid=105503.R.357841007819607`

策略链接选择器：`a[href*="sid="]`
